# Macework Creative Landing Page

Macework Creativ için modern, premium ve SaaS odaklı yeni nesil landing page.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4
- **Icons**: Lucide React
- **Utilities**: clsx / tailwind-merge (cn helper)

## Özellikler

- 🚀 Fully Responsive Premium Tasarım
- 🌑 Modern Karanlık Mod Teması
- 📦 Bileşen Tabanlı Mimari
- 🏗️ Merkezi İçerik Yönetimi (`src/content/site-content.ts`)
- ⚡ Optimize Edilmiş Performans & SEO
- 🏗️ Mega Menu (Desktop) & Drawer Nav (Mobile)
- 📝 Lead Generation Formu (Mock Submit)

## Kurulum ve Çalıştırma

1. Bağımlılıkları yükleyin:
```bash
npm install
```

2. Geliştirme sunucusunu başlatın:
```bash
npm run dev
```

3. Production (Canlı) sürümü build edin:
```bash
npm run build
```

4. Build çıktısını test edin:
```bash
npm run start
```

## Proje Yapısı

- `/src/app`: Sayfalar ve global stiller
- `/src/components`: UI bileşenleri
- `/src/content`: Tüm statik metinler ve veriler
- `/src/lib`: Yardımcı fonksiyonlar
- `/public`: Görsel ve marka varlıkları
